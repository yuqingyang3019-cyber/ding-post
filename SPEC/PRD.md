# 污水处理药剂价格机器人 PRD

## 目标

首版提供一个钉钉群机器人回调服务，收到钉钉请求后返回污水处理药剂价格早报 Markdown，便于群内快速查看 PAC、PAM、复合碳源、片碱、污泥处置参考价和行业早报。

## 首版范围

- 接收钉钉机器人回调请求。
- 校验请求必须是 JSON 对象。
- 返回钉钉 Markdown 消息，标题为“污水处理药剂价格早报”。
- 使用代码内结构化假数据 `MOCK_PRICE_JSON` 渲染表格和提示区。
- 提供 `/health` 健康检查接口。
- 支持通过环境变量启用钉钉签名校验。

## 非目标

- 不接入真实行情源。
- 不提供后台管理页面。
- 不接入数据库。
- 不引入合同业务、报价单上传、LLM 合同识别或钉盘上传能力。

## 迭代路线

1. MVP：固定假数据、钉钉 Markdown 回调、FC3 部署。
2. 支持按关键词返回不同内容，例如“价格”“污泥”“早报”。
3. 将 `MOCK_PRICE_JSON` 替换为数据库或管理表单维护。
4. 接入真实行情源或内部采购价格，增加更新时间、来源标注和异常兜底提示。
